# Birthday Card Website 🎂💖
A simple and creative birthday card website built using **HTML, CSS, and JavaScript**.  
This project was special to me because it was my **first HTML project**, created for my close friend as a digital birthday surprise.

---

## ✨ Features
- Personalized birthday message
- Beautiful and colorful UI
- Animations and interactive effects
-  Responsive design
-  Surprise birthday elements

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---
## 🚀 How to Run

1. Clone the repository
git clone <your-repository-link>

2. Open the project folder
cd Birthday-Card

3. Run the project by opening index.html in your browser.
